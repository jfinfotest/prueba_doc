# Bienvenido a Fusiondoc-Next

Fusiondoc es un motor de documentación estática impulsado por **Next.js 16 (App Router)** y **GitHub**. 

Esta plataforma está diseñada para ser extremadamente rápida, hermosa y fácil de usar, permitiéndote gestionar todo tu contenido directamente desde tu repositorio de GitHub sin necesidad de bases de datos ni configuraciones complejas.

## Características Principales

- ⚡ **Next.js 16 + React 19:** El rendimiento más alto con las últimas tecnologías.
- 🎨 **Diseño Premium:** Estética moderna, fluida y con soporte nativo para **Modo Oscuro**.
- 🛠️ **Componentes MDX:** Pestañas de código interactivas, terminales animadas y mucho más.
- 🐙 **Integración con GitHub:** Sincronización automática de contenido.
- 🔍 **SEO Optimizado:** Mejoras automáticas para motores de búsqueda.
