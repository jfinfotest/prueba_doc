# Instalación

Sigue estos simples pasos para instalar y compilar el proyecto en tu máquina.

```bash
git clone https://github.com/tu-usuario/fusiondoc-next.git
cd fusiondoc-next
npm install
npm run dev
```

Una vez que el servidor inicie en `http://localhost:3000`, la aplicación leerá los documentos directamente desde este repositorio usando la API de GitHub.
